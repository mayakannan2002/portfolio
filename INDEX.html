Project 2

LIBRARY MANAGEMENT SYSTEM

The Library Management System is a web-based platform designed to streamline library operations by allowing administrators to manage book inventories and track borrowing records. The system ensures efficient handling of library resources and simplifies the user experience with a clean, interactive interface.

  • Input Form: The administrator can input details such as Book Title, Author, Publication Year, and ISBN through a user-friendly form.
  • Dynamic Inventory Display: The added book details are dynamically updated in a table using JavaScript, making it easy to track library holdings in real time.
  • Edit Functionality: Admins can select a book entry to edit by clicking an "Edit" button, which automatically fills the form with the selected book's details for quick modification.
  • Delete Functionality: Admins can remove a book record from the inventory by clicking a "Delete" button.
  • Borrow and Return Tracking: Additional features allow the admin to track when a book is borrowed, by whom, and its return status.

Project 2

STUDENT MANAGEMENT SYSTEM

The Student Management System is a web-based application designed to manage student data efficiently. This system enables an administrator to perform key actions like adding, editing, viewing, and deleting student records. The project focuses on providing a user-friendly interface and ensuring that the data is organized and accessible.

  • Input Form: The user enters student details like Name, Date of Birth, and Place into a form.
  • Dynamic Table Update: Once submitted, the form data is displayed dynamically in a table using JavaScript.
  • Edit Functionality: Admin can click on an "Edit" button next to each record, which populates the form with the selected student's details for modification.
  • Delete Functionality: Admin can click on a "Delete" button to remove the corresponding student record from the table.
